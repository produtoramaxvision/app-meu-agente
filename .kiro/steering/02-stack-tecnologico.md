# Stack Tecnológico do Meu Agente

## Frontend

### Framework e Build
- **React 18.2.0**: Framework principal
- **TypeScript 5.0+**: Linguagem obrigatória para todo código
- **Vite 4.0+**: Build tool e dev server

### Estilização
- **Tailwind CSS 3.0+**: Framework CSS (NUNCA escrever CSS customizado)
- **ShadcnUI v4**: Biblioteca de componentes UI
- **Radix UI**: Base dos componentes shadcn/ui

### Bibliotecas Principais
- **React Router**: Roteamento client-side
- **React Hook Form**: Gerenciamento de formulários
- **Zod**: Validação de schemas
- **TanStack Query**: Gerenciamento de estado do servidor
- **Recharts**: Visualização de dados
- **Lucide React**: Ícones
- **Sonner**: Notificações toast

## Backend

### Supabase
- **PostgreSQL**: Banco de dados principal
- **Edge Functions**: Funções serverless
- **Row Level Security (RLS)**: Segurança de dados
- **Realtime**: WebSocket para atualizações em tempo real

### Integrações
- **n8n**: Automação e webhooks para Chat IA
- **Evolution API**: WhatsApp para Agente SDR
- **Stripe**: Pagamentos e assinaturas

## DevOps
- **Vercel**: Deploy e hosting
- **GitHub Actions**: CI/CD
- **Supabase CLI**: Gerenciamento de banco
